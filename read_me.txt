This file contains the instructions in order to update the directory into Git hub and push and pull information. In addition, it includes the basic commands of Git.

Git commands:

git init [project name] # Creates a new local repository with the specified name

git clone [url]# Downloads a project and its entire version history

git status #Lists all new or modified files to be comited

git diff #Shows file differences not yet staged

git add [file]# Moves a file to the staging area in preparation for commiting.
